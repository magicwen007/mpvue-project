# mpvue-project

## 简介

> 这是一个mpvue的实例工程

## 安装与运行

* 下载工程


    git clone https://github.com/magicwen007/mpvue-project.git


* 安装依赖库


    npm install


* 运行实例


    npm run dev


* [微信开发者工具配置](http://mpvue.com/mpvue/quickstart/#3-mpvue)
